- Yêu cầu trên máy:
	+ Máy phải có môi trường nodeJS
	+ Máy phải cài đặt phần mềm XAMPP

- Các bước thiết lặp môi trường để chạy project:
			--- Tạo database ---
	+ Mở cổng Apache và MySQl (vì database sử dụng cho project là MySQL)
	+ Truy cập vào phpByAdmin
	+ Import file 'wallet.sql'

	 		--- Cấu hình và chạy project ---
	+ Dùng editer bất kỳ (khuyến khích sử dụng Visual Studio Code)
	+ Mở terminal trỏ vào thư mực 'Source' - nơi chứa toàn bộ source code của project
	+ Trên terminal chạy lênh: 'npm i' - download các package cần thiết
	+ Trên terminal tiếp tục chạy lệnh: 'npm start' - dùng để mở port 3000 để chạy project
